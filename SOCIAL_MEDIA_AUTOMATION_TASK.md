**TASK: Social Media Automation Workflow**

Execute the following steps in order:

### Step 1: TikTok Setup
1. Navigate to https://www.tiktok.com
2. Log in to the account (wait for manual login if 2FA is required)
3. Go to profile settings
4. Change display name to: "ours money zone"
5. Save changes and confirm

### Step 2: Create Video Content
Create a 15-second vertical video (1080x1920) with:
- Black background
- White text displaying: "You waste $30 a month on shit. Stop. Get NordVPN for 10 bucks. Link below."
- Text centered, large, and readable
- Duration: exactly 15 seconds
- Voice-over: Use your voice, slow and cold delivery, ending with "Peace"
- Save as: ./tmp/videos/tiktok_video.mp4

### Step 3: Post to TikTok
1. Click the create/upload button (+ icon) on TikTok
2. Upload the video file: ./tmp/videos/tiktok_video.mp4
3. Wait for video to process
4. Add caption: "No fun. No emotion, only the wind. Fine."
5. Add affiliate links in description:
   - NordVPN affiliate link
   - Abacus AI affiliate link (if applicable)
6. Post the video
7. Confirm post was successful

### Step 4: Setup Scheduler (Every 45 minutes)
- Set up automated posting every 45 minutes
- Repeat Step 3 at each interval
- Use the same video and caption each time

### Step 5: Post to Twitter
1. Navigate to https://twitter.com or https://x.com
2. Log in if needed
3. Click compose tweet
4. Upload the same video: ./tmp/videos/tiktok_video.mp4
5. Add caption: "No fun. No emotion, only the wind. Fine."
6. Add affiliate links:
   - NordVPN affiliate link
   - Abacus AI affiliate link (if applicable)
7. Post the tweet immediately
8. Confirm post was successful

### Step 6: InVideo AI - Create Variations
Navigate to https://invideo.ai and create 3 video variations:

**Variation 1:**
- Text: "You're wasting $30 monthly. Stop. Get NordVPN for $10. Link below."
- Save as: ./tmp/videos/invideo_variation_1.mp4

**Variation 2:**
- Text: "Stop wasting $30 a month. Get NordVPN for 10 bucks. Link below."
- Save as: ./tmp/videos/invideo_variation_2.mp4

**Variation 3:**
- Text: "You waste $30 monthly on crap. Stop. NordVPN for $10. Link below."
- Save as: ./tmp/videos/invideo_variation_3.mp4

### Step 7: Rotate Variations
- Post variation 1 to TikTok and Twitter
- Wait 45 minutes
- Post variation 2 to TikTok and Twitter
- Wait 45 minutes
- Post variation 3 to TikTok and Twitter
- Rotate back to original, then repeat cycle

### Step 8: Payment Monitoring
When money hits (from affiliate earnings - NordVPN, Abacus AI, InVideo):
